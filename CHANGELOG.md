# LazyFisher 合并版 · 更新日志

合并脚本将以下模块整合为单一油猴脚本，按版本递增发布，**旧版文件保留不删**。

| 模块 | 说明 |
|------|------|
| LazyFisher 信息增强 | 鱼护统计、消息页增强、船钓辅助、升级经验等 |
| lazyfisher 辅助增强（公开版） | 鱼群排序、水层标签、钓行统计、商店增强等 |
| LazyFisher 环境显示 | 水流/湿度/降雨、附近玩家、软封状态 |

**当前最新版：** `LazyFisher合并版V1.10.user.js`

---

## 如何回顾某一版改了什么

1. **看本文件** — 每个版本下方的条目即摘要。
2. **看 `ARCHITECTURE.md`** — 代码模块地图与跨模块钩子。
3. **对比相邻版本文件** — 在 Cursor 中选中两个 `LazyFisher合并版V*.user.js`，右键 **Compare Selected** 查看代码 diff。
4. **看文件头 `@version`** — 应与文件名版本一致。

### 版本号规则

- 预发布：`V0.1` … `V0.10`
- **`V0.10` 后进位为 `V1.0`**
- 之后：`V1.1` … `V1.10` → `V2.0`（满十进一）
